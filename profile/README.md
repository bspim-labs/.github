# Bluestone PIM Labs

A community space for builders working on and for [Bluestone PIM](https://www.bluestonepim.com).

Labs is where side projects, experiments, and community tools live. These are not official Bluestone products. They are built by people who use and work with Bluestone PIM and want to push it further. Everything here is open, forkable, and built in the open.

## Projects

| Project | Description | Status |
|---|---|---|
| [MCP Server](https://bluestone-mcp-unofficial.vercel.app/connect) | Connect Claude, Cursor, and ChatGPT to your Bluestone PIM organisation | Live |

## Get involved

Built something on or for Bluestone PIM? It belongs here.

Visit [bluestone-labs-landing.vercel.app](https://bluestone-labs-landing.vercel.app) to see all projects and learn how to submit yours. No gating process: open a pull request, add your entry to `projects.json`, and the page updates automatically.
